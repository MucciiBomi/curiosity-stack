# India Proxy

Activates the India Proxy research agent.

The agent will:
1. Clarify the value chain layer if needed (one question maximum)
2. Research the global company or theme to understand the actual business model
3. Search Tracxn, Inc42, Screener.in, NASSCOM, and general web sources for Indian equivalents
4. Validate each candidate against red flag criteria
5. Return a structured shortlist with proxy pattern, access method, validation signals, and watch triggers

This is autonomous research — you do not need to run any searches yourself.

Check your connected sources (Drive, Notion, Gmail) first for any prior research on this topic.

*All outputs are research summaries only. Not investment advice. Not a recommendation to buy, sell, or hold any security.*
